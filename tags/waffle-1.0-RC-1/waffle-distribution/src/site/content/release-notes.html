  • [WAFFLE-38] - Introduce ErrorMessage type in ErrorsContext

Migrating from 1.0-beta-1 to 1.0-rc-1

The following issues will break backward compatibility:

  • Waffle Core artifactId changed from waffle to waffle-core [WAFFLE-31]
  • Waffle Monitor dependencies added [WAFFLE-10]
  • Mocks moved to Waffle Mock (artifactId waffle-mock) and upgraded to using jMock 2. [WAFFLE-29] [WAFFLE-30]
  • Waffle ValueConverterFinder replaced Ognl TypeConverter as pluggable interface [WAFFLE-32] [WAFFLE-33]
  • Ognl-based implementations renamed and/or moved [WAFFLE-34] [WAFFLE-35]
  • Validation errors context changed and simplified [WAFFLE-38] [WAFFLE-39]
  • Refactored annotations [WAFFLE-47] [WAFFLE-48]
  • Registrar interface enhanced [WAFFLE-11] [WAFFLE-45]
